# Scoreboard-API
A simple Scoreboard API using reflection

# Usage

```java
NMSBoard board = new NMSBoard(bp.getPlayer(), "§aTest §bScoreboard");
board.set(player, 15, "§6§m--------------------");
board.set(player, 14, "§eHello world!");
board.set(player, 13, "");
board.set(player, 12, "It works!");
```
